Author Guideline

* The manuscript should be type set to A4 two columns with Times Roman Font with Font Size 12, Spacing 1.5  
* The maximum numbers of pages not restricted, the file should be word document
* If any, queries please feel free to communicate with the conveners and also visit our conference 
website: www.mitconference.in
The paper can be send to,
Email: contact@mitconference.in
Paper presentation by the author must be done through online video conferencing only.
